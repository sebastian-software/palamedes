# Remix v3 Subdomain Example

This example proves the Remix v3 server integration with a subdomain-derived
locale (`de.lvh.me` -> `de`).

Run it locally:

```sh
pnpm --filter @palamedes/example-remix-subdomain build
pnpm --filter @palamedes/example-remix-subdomain start
```

Then open <http://de.lvh.me:4062/>.
