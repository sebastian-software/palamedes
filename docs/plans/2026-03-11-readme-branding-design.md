# README Branding Design

Date: 2026-03-11

## Goal

Add lightweight Sebastian Software branding to the root README and all package READMEs without overwhelming the package identity.

## Decisions

- Keep package names and product copy as the primary visual focus.
- Add a `Sponsored by Sebastian Software` badge near the existing badges at the top of each README.
- Add the Sebastian Software logo near the copyright area in the footer, directly above the license line.
- Keep the license line simple and consistent across all READMEs.

## Badge Pattern

- Text: `Sponsored by Sebastian Software`
- Link target: `https://oss.sebastian-software.com/`
- Applied to root README and every package README

## Footer Pattern

- Clickable Sebastian Software logo:
  - Image: `https://sebastian-brand.vercel.app/sebastian-software/logo-software.svg`
  - Link: `https://oss.sebastian-software.com/`
- Followed by:
  - `MIT © 2026 Sebastian Software`

## Constraints

- No other README copy changes in this pass
- No header logo
- Keep the visual treatment quiet and consistent with npm landing pages
