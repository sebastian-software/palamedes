# Homepage Visual Exploration

These generated mockups are planning artifacts for the Palamedes homepage
direction. They are intentionally not implementation-ready designs.

The variants explore different visual directions for a public page that can
explain both layers:

- Palamedes as the open-source JavaScript i18n foundation
- Palamedes+ as the managed AI translation and language-governance layer

The generated text inside the images is illustrative only. Any final homepage
copy, package references, install commands, product claims, and pricing language
must be checked against the repository docs before implementation.
