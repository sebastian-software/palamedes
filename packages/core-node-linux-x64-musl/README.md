# @palamedes/core-node-linux-x64-musl

Platform-specific native bindings for `@palamedes/core-node` on Linux x64 with
musl libc, such as Alpine Linux.

Install [`@palamedes/core-node`](https://www.npmjs.com/package/@palamedes/core-node)
instead of this package directly. This package is loaded internally as an
optional dependency.
