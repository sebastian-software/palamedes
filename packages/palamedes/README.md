# palamedes

Reserved package name for the Palamedes toolchain.

This package is currently a lightweight placeholder while the scoped packages remain the supported entry points:

- `@palamedes/cli`
- `@palamedes/extractor`
- `@palamedes/transform`
- `@palamedes/runtime`
- `@palamedes/vite-plugin`
- `@palamedes/next-plugin`

The package name is intentionally published now so the project can keep the canonical npm identity for future use.
